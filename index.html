Methodology

TON unlock data is sourced from vesting-wallets.json gathered using DTON GraphQL API. The chart displays cumulative token unlocks over time based on each wallet's vesting schedule. Each wallet has specific parameters: start time, total duration, unlock period, cliff duration, and total amount. Unlocking follows a linear schedule after the cliff period, with tokens being released periodically. This data includes vesting wallets but does not include the TON Believers Fund.
